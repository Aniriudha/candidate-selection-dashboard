# Candidate Selection Dashboard

A full-stack Candidate Selection Dashboard that evaluates candidates using AI-based scoring and visualizes rankings in a modern React dashboard.

---

## Features
- Candidate database (MySQL)
- AI-based evaluation (mocked)
- Ranking system
- Leaderboard dashboard
- Score visualization
- Candidate profiles
- Skill heatmap

---

## Tech Stack
**Frontend**
- React + Vite
- Mantine UI

**Backend**
- Node.js
- Express.js
- MySQL
- Faker.js
